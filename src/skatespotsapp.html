About Work

Street Skate Spots is een React Native app die je verschillende street spots in Rotterdam laat zien, die perfect zijn voor het skateboarden. Er is een map waar je de verschillende spots kan zien d.m.v markers. Druk op een marker voor meer informatie over de spot en om een beeld te krijgen van hoe de spot eruit ziet. In de skatespots lijst kan je de verschillende spots die op de map staan overzichtelijk zien in een lijst, en ze favorieten. Daarnaast is er ook een darkmode. Ik heb dit gemaakt voor een schoolopdracht maar ben daarna er aan verder gaan werken omdat ik erg graag met React Native werk.
